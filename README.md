# p4r

Yet another command-line tools to build machine image on cloud platforms.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'p4r'
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```
$ gem install p4r
```
